# クイズミリオネア風のアプリの作成

- URL:https://quiz-millionare.netlify.app

### 概要

- 4択から選択して、正解・不正解で音声がなる。

### 機能要件

- 開始ボタン:クイズを開始する
- 選択: 時間内にクイズを選択する
- 終了: 正解数に応じた得点の表示
- クイズ開始・正解・不正解で音声あり
- ミュート機能あり
